# Privacy Policy for Pet Planner

Effective Date: December 2025

Pet Planner is a student-developed iOS application created for educational purposes.

## Information Collection and Use
Pet Planner does not collect, store, or transmit any personal data to external servers.
All data entered into the app (such as pet names, appointments, and clinic information)
is stored locally on the user’s device using Apple’s Core Data framework.

## Data Storage
All data remains on the user’s device and is not shared with third parties.
The app does not require user accounts or internet-based data storage.

## Permissions
Pet Planner does not request access to sensitive device permissions such as camera,
contacts, microphone, or location services.

## Third-Party Services
This app does not use third-party analytics, advertising, or tracking services.

## Changes to This Privacy Policy
This Privacy Policy may be updated if the app functionality changes.

## Contact
This application was created as part of a college course project.
If you have any questions, please contact the project team via GitHub.
